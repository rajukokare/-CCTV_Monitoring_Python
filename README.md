# 📡 CCTV Uptime Monitoring Automation Using Python

## 📌 Problem Statement
In my current role, monitoring 14 CCTV cameras was a manual task performed every 3 hours.  
This process was:
- Time-consuming  
- Prone to human error  
- Lacked precise downtime tracking  

## 🚀 Solution
I developed an automation system using Python that:

- Automatically pings each camera's IP address  
- Runs every 1 hour using a scheduled loop  
- Logs camera status (Online/Offline) with timestamps  
- Stores data in a CSV file for easy tracking and reporting  

## 🛠 Tech Stack
- Python  
- Libraries: os, csv, datetime  

## 📊 Results & Impact
- ✅ Eliminated manual monitoring effort  
- ✅ Improved accuracy to near 100%  
- ✅ Enabled precise downtime tracking  
- ✅ Simplified reporting using CSV data (Excel / Power BI ready)
